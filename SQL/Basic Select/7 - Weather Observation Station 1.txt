Query a list of CITY and STATE from the STATION table.

Input Format

The STATION table is described as follows:

	STATION
	
Field             Type
ID                Number
City              Varchar2(21)
State	          Varchar2(3)
LAT_N	          Number
LONG_W   	      Number

where LAT_N is the northern latitude and LONG_W is the western longitude.

Query :-

select city,state from station;
